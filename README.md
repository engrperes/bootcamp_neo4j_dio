# bootcamp_neo4j_dio

Neste repositório serão armazenados os projetos práticos propostos pelo Bootcamp da Neo4j dentro da DIO.

## Entrega 1 - Modelagem de Dados em Grafos de um Serviço de Streaming

Projeto de um banco de Grafos de uma plataforma de Streaming onde foram utilizados nomes de usuários, atores, diretores e filmes fictícios.
Para criação deste projeto foi utilizado a ferramenta arrows.app. 
O projeto é constituído de 51 nós e 78 relacionamentos.
- Nós utilizados: User, Actor, Directore Movie e Serie
- Relacionamentos utilizados: Watched, Acted_in e Directed_in,
